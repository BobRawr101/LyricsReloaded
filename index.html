LyricsReloaded

A new aproach to MusicBee's lyrics plugin

Download this project as a .zip file Download this project as a tar.gz file

Welcome to the new Lyrics plugin for MusicBee.

It uses a new approuch to loading lyrics warom various sources. Instead of hardcoding the supported pages in code, this plugin loads simple YAML configurations from a folder which contain enough information to allow te plugin to read lyrics from pages.

Downloads

Version 1.0 is available: download mb_LyricsReloaded.dll

For bleeding edge builds you'll need to download the sources and build it yourself.

Authors and Contributors

This plugin is developed by @quickwango and is inspired by the original Lyrics plugin developed by the MusicBee developer.
